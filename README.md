# Hand Gesture Recognition System

## 📌 Description
This project detects hand gestures (Yes, No, Hello) using a CNN model and real-time webcam feed.

## 🚀 Features
- Real-time gesture detection
- CNN-based deep learning model
- Voice output using pyttsx3
- Flask web interface

## 🛠️ Technologies Used
- Python
- OpenCV
- TensorFlow/Keras
- Flask

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   py -3.10 app.py

3. Open browser:
   http://127.0.0.1:5000

## ⚠️ Note
- Dataset and model are not uploaded due to size limits.
- You can train the model using train.py
